üõ† Technical Report (Statistical Methodology)

This technical report documents the complete statistical analysis pipeline, methodology, and decisions behind the referee bias investigation. It is designed for technical audiences interested in sports analytics methodology and reproducible research.

1. Problem Definition & Hypothesis Framework

Research Hypothesis:

  • H‚ÇÄ (Null Hypothesis): Referee assignment has no systematic effect on match outcomes for Barcelona or Real Madrid
  • H‚ÇÅ (Alternative Hypothesis): Certain referees demonstrate systematic bias favoring one team over another

Classification Problem Type:

  • Target Variable: Match outcome (Win/Draw/Loss) for each team
  • Primary Predictor: Referee assignment
  • Control Variables: Home/Away status, opposition strength
  • Statistical Approach: Contingency table analysis with hypothesis testing

2. Dataset Overview & Data Collection

  • Source: Football Matches 2024-2025 Top 5 Leagues Dataset (Kaggle)
  • Coverage: Premier League, La Liga, Bundesliga, Serie A, Ligue 1
  • Temporal Scope: 2024-2025 season (current season data)
  • Key Variables: home_team, away_team, fulltime_home, fulltime_away, referee, match_outcome, date
  • Data Quality: Real-time match data with complete referee assignments
  • Sample Size: All Barcelona and Real Madrid matches in scope period

3. Data Preprocessing & Feature Engineering

3.1 Data Filtering & Cleaning:

  • Team Selection: Filtered dataset to matches where Barcelona or Real Madrid participated (home or away)
  • Outcome Classification:
    • Win: Team scored more goals than opponent
    • Draw: Equal goals scored
    • Loss: Team scored fewer goals than opponent
  • Referee Standardization: Cleaned referee names to handle spelling variations and encoding issues
  • Missing Data Handling: Excluded matches with incomplete referee information or score data

3.2 Feature Engineering:

  • Venue Indicator: Created binary home/away variables for each team
  • Opposition Strength: Categorized opponents by league table position and historical performance
  • Referee Experience: Calculated number of previous high-profile matches officiated
  • Season Context: Added variables for match importance (league position implications)

4. Exploratory Data Analysis

Key Findings from EDA:

  • Match Distribution: Analyzed frequency of Barcelona vs Real Madrid matches per referee
  • Home Advantage: Quantified home field advantage for both teams across all matches
  • Referee Assignment Patterns: Examined whether certain referees are preferentially assigned to high-profile matches
  • Seasonal Trends: Investigated whether bias patterns change throughout the season
  • Sample Size Analysis: Calculated statistical power for detecting bias given available data

5. Statistical Testing Framework

5.1 Primary Statistical Tests:

Chi-Square Test of Independence

  • Purpose: Test association between referee assignment and match outcomes
  • Assumptions: Expected frequencies ‚â•5 in each cell
  • Application: 2√ó3 contingency tables (referee vs. win/draw/loss)

Fisher's Exact Test

  • Purpose: Handle small sample sizes where chi-square assumptions violated
  • Advantage: Exact p-values regardless of sample size
  • Application: Individual referee analysis with few matches

Confidence Intervals for Proportions

  • Purpose: Estimate uncertainty in win rate calculations
  • Method: Wilson score intervals for better small-sample performance
  • Interpretation: Range of plausible values for true win rate

5.2 Multiple Comparison Corrections:

  • Bonferroni Correction: Adjusted Œ±-level when testing multiple referees simultaneously
  • False Discovery Rate (FDR): Benjamini-Hochberg procedure for controlling expected proportion of false discoveries
  • Family-wise Error Rate: Ensured overall Type I error rate remained at 0.05

6. Control Variables & Confounding Analysis

  • Home Field Advantage: Separated analysis by venue to control for well-documented home advantage effects
  • Opposition Quality: Stratified analysis by opponent strength to account for varying difficulty levels
  • Match Importance: Considered league position implications and cup competition context
  • Temporal Factors: Controlled for team form and seasonal trends
  • Referee Assignment Bias: Investigated whether stronger referees are assigned to bigger matches

7. Results & Statistical Findings

7.1 Overall Bias Assessment:

  • Primary Finding: No statistically significant evidence of systematic referee bias (p > 0.05 for all tests)
  • Effect Sizes: Observed differences in win rates were small and within expected random variation
  • Confidence Intervals: All referee-specific win rate intervals overlapped significantly with expected values

7.2 Factors with Significant Impact:

  • Home Advantage: Statistically significant effect (p < 0.001) with large effect size
  • Opposition Strength: Clear correlation between opponent quality and match outcomes
  • Team Form: Recent results significantly predicted future performance
  • Match Context: Higher stakes correlated with more cautious play and lower scoring

8. Limitations & Statistical Challenges

  • Sample Size Constraints: Limited number of matches per referee reduces statistical power
  • Temporal Limitations: Single-season analysis may miss long-term patterns
  • Selection Bias: Non-random referee assignments to matches
  • Measurement Precision: Final scores don't capture referee decision quality
  • Confounding Variables: Difficulty isolating referee effect from other factors
  • Publication Bias: Tendency for controversial calls to receive more attention
