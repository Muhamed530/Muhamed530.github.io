🏦 Can AI Predict Who Will Default on Their Loan?

Every day, financial institutions face a critical question: should they approve this loan application? Behind every decision lies the risk of default – borrowers who can't repay what they've borrowed. But what if machine learning could help predict these outcomes before they happen?

Using historical data from LendingClub containing over 2.26 million loan records, I built classification models to predict loan defaults. This project explores whether borrower characteristics and loan features can reliably forecast financial risk.

Key Questions Explored:

  • Can we accurately predict which loans will default based on borrower information?
  • Which features are most predictive of loan default risk?
  • How do different classification algorithms perform on financial risk assessment?

Key Takeaways

  • 🎯 Predictability: Loan defaults can be predicted with reasonable accuracy using borrower and loan characteristics
  • 📊 Feature Importance: Credit grades, interest rates, and debt ratios are top predictors
  • 🤖 Model Selection: Random Forest models excel at capturing complex relationships in financial data
  • ⚖️ Ethical AI: Financial ML requires careful attention to fairness and transparency

🛠 Technical Report

This project used Random Forest and Logistic Regression models to predict loan defaults using LendingClub data with over 2.26 million records.
